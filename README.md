# coc-project-student-score-analyzer
This C program analyzes student quiz scores. It uses a 1D Array and for loops to take $N$ inputs1. Logic is modularized into Functions to calculate the Mean (Average), Highest, and Lowest scores2. This project synthesizes C concepts (Arrays, Functions) and Maths (Mean, Data Analysis) 3. Submitted via a public GitHub repo with a detailed README.md
